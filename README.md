# SOLVE STATION LABS AI AGENT WEBINAR

## Overview

This repository contains materials and projects for the Solve Station Labs webinar on building AI-powered applications and intelligent agents using LangChain, LangGraph, and large language models (LLMs). The webinar demonstrates practical implementation patterns for integrating AI capabilities into modern applications.

## What This Webinar Covers

### Core Concepts

- **LangChain Framework**: Understanding the fundamentals of LangChain for building LLM-powered applications
- **LangGraph**: Creating stateful, multi-actor applications with complex workflows modeled as graphs
- **AI Agents**: Building autonomous systems that can perceive, decide, and act to achieve specific goals
- **Prompt Engineering**: Techniques for effective communication with language models
- **Conversation Management**: Implementing context-aware conversational interfaces
- **Agent Orchestration**: Coordinating multiple AI agents to accomplish complex tasks

### Technologies Demonstrated

- Node.js and TypeScript for type-safe development
- Google Gemini AI for language model capabilities
- Express.js for REST API development
- LangGraph for agent workflow management
- PostgreSQL with Drizzle ORM for data persistence
- Google Cloud services integration (Drive, Calendar)
- Authentication and authorization patterns
- Real-time communication with WebSockets

## Projects Included

### 1. LLM Agent Starter

A foundational project demonstrating basic LangChain concepts including:
- Integration with Google's Gemini AI model
- Simple chain execution for question-answering
- Prompt template usage
- Conversation context management
- Streaming responses for real-time output

This starter project serves as an introduction to building AI-powered applications with minimal complexity.

### 2. Simple LangChain Tutorial

An educational demonstration focusing on core LangChain patterns:
- Prompt engineering techniques
- Chain composition and execution
- Conversational AI assistant implementation
- Memory and context handling
- Streaming response management

Designed for hands-on learning during the webinar sessions.

### 3. Study-Buddy

A comprehensive production-ready application showcasing advanced AI agent implementation:

**Features:**
- Intelligent content simplification using multi-agent architecture
- PDF generation and document processing
- Google Drive integration for file management
- AI-generated study schedules with Google Calendar sync
- Interactive AI tutor chat interface
- Automated quiz generation
- Secure authentication with JWT and Google OAuth
- Swagger API documentation

**Agent Architecture:**
- Research Agent: Analyzes and simplifies learning materials
- Compiler Agent: Generates formatted PDFs and study guides
- Scheduler Agent: Creates optimized study plans
- Tutor Agent: Provides interactive learning assistance

This project demonstrates enterprise-grade patterns for building sophisticated AI agent systems.

## Prerequisites

To work with the projects in this repository, you will need:

- Node.js version 18 or higher
- pnpm package manager
- Google API credentials for Gemini AI
- Google Cloud project credentials (for Study-Buddy)
- PostgreSQL database (for Study-Buddy)
- Basic understanding of TypeScript and Node.js

## Getting Started

Each project contains its own README with detailed setup instructions. Navigate to the respective project directory and follow the installation and configuration steps.

### Quick Start

1. Choose a project directory
2. Install dependencies: `pnpm install`
3. Configure environment variables in `.env` file
4. Run the application according to project-specific instructions

## Learning Path

The projects are ordered by complexity:

1. **Start with LLM Agent Starter** - Learn basic LangChain concepts and simple LLM integration
2. **Proceed to Simple LangChain Tutorial** - Explore conversation management and prompt patterns
3. **Advance to Study-Buddy** - Study production patterns, multi-agent systems, and enterprise architecture
